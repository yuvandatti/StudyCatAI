# 🎨 Brand Guidelines – StudyCat AI

---

# 🐱 Brand Name

## Brand Name with Quotation
**StudyCat AI**  
> "Focus. Grow. Repeat."

---

## ✅ Acceptable Usage
- Always write as: **StudyCat AI**
- Capital “S” and “C” mandatory
- Use full name in formal contexts
- Can shorten to **StudyCat** in UI labels

---

## ❌ Unacceptable Usage
- studycat / STUDYCAT / Study cat
- SCAT AI (no abbreviations)
- Changing spacing or capitalization

---

# 🐱 Logo Usage

## Clear Space
- Maintain spacing equal to **height of “S”**
- No elements should overlap logo boundary

---

## Size
- Minimum size:
  - Web: 32px height
  - Mobile: 24px height
- Must remain clearly visible and readable

---

## ✅ Logo DOs
- Use on dark background (preferred)
- Maintain original proportions
- Use high-resolution assets
- Place in header or top-left

---

## ❌ Logo DON’Ts
- Do not stretch or distort
- Do not change colors randomly
- Do not add shadows/glows externally
- Do not place on cluttered background

---

# 🎨 Color Palette

## Primary Colors
- **Midnight Black** → #0F0F0F (background)
- **Neon Blue** → #4FC3F7 (primary highlight)
- **Electric Purple** → #9B5DE5 (accent)

---

## Secondary Colors
- **Soft Green** → #4CAF50 (success)
- **Warm Orange** → #FF9F1C (warning)
- **Soft Red** → #FF4D6D (error)

---

## Neutral Colors
- **Dark Gray** → #1E1E1E
- **Medium Gray** → #666666
- **Light Gray** → #CCCCCC
- **White** → #FFFFFF

---

# 🧩 UI Elements

## 🔘 Buttons

### Primary Buttons
- Solid neon gradient (blue → purple)
- Rounded corners (12px–16px)
- Glow effect on hover
- Large size for main actions

### Secondary Buttons
- Outline style
- Transparent background
- Subtle hover fill

---

## 🧾 Cards
- Dark background (#1E1E1E)
- Rounded corners (16px)
- Soft shadow or glow
- Padding: 16px–24px

---

## ⌨️ Inputs
- Dark background
- Light border (#666)
- Focus state → neon glow
- Placeholder text: subtle gray

---

## 📊 Progress Indicators
- Circular progress (for timer)
- Smooth animated transitions
- Use primary neon colors

---

# 🎯 Iconography

- Use simple, minimal icons
- Line-based icons preferred
- Consistent stroke width
- Examples:
  - Timer ⏱️
  - Streak 🔥
  - Tasks 📝

---

# 🔤 Typography

## Font Style
- Primary: **Poppins / Inter**
- Secondary: **Roboto**

---

## Font Hierarchy
- Headings: Bold, large
- Subheadings: Medium weight
- Body: Regular, clean
- Buttons: Slightly bold

---

## Rules
- Maintain consistent spacing
- Avoid too many font types
- Ensure readability in dark mode

---

# ✨ Motion & Animation

- Smooth transitions (0.2s – 0.4s)
- No harsh or sudden movements
- Use subtle glow animations
- Micro-interactions:
  - Button hover
  - Task completion
  - Cat reactions

---

# 📱 Layout Guidelines

- Mobile-first design
- Use grid system (8px spacing rule)
- Maintain consistent margins/padding
- Avoid overcrowding UI

---

# 🧠 UX Principles

- One primary action per screen
- Minimal distractions
- Fast interaction response
- Clear visual hierarchy

---

# 🐱 Brand Personality

- Friendly 🤝
- Motivating 🔥
- Calm 🧘
- Slightly playful 😸
- Never childish

---

# 📢 Content & Tone

- Short sentences
- Encouraging language
- Action-driven phrases:

  - “You got this”

# 🌐 Web Standards

- Fast loading UI

- Keyboard navigable
- SEO-friendly structure
